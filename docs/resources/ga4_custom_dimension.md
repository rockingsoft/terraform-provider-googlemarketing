# googlemarketing_ga4_custom_dimension

Typed GA4 custom dimension resource.

Manages parameter name, display name, scope, and description.

## Import

Import with `{property_id}.{custom_dimension_id}`:

```bash
terraform import googlemarketing_ga4_custom_dimension.example 123456789.987654321
```

The full GA4 Admin name is also accepted, for example `properties/123456789/customDimensions/987654321`.
