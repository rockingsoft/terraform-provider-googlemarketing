# googlemarketing_ga4_property

Typed GA4 property resource.

Manages display name, time zone, currency code, and exposes the numeric `property_id`.

## Import

Import with `{account_id}.{property_id}`:

```bash
terraform import googlemarketing_ga4_property.example 123456789.987654321
```

The full GA4 Admin name is also accepted, for example `properties/987654321`.
