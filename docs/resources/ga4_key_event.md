# googlemarketing_ga4_key_event

Typed GA4 key event resource.

Manages a GA4 key event by `event_name`.

## Import

Import with `{property_id}.{key_event_id}`:

```bash
terraform import googlemarketing_ga4_key_event.example 123456789.987654321
```

The full GA4 Admin name is also accepted, for example `properties/123456789/keyEvents/987654321`.
