# googlemarketing_ga4_web_data_stream

Typed GA4 web data stream resource.

Manages display name and default URI, and exposes the GA4 measurement ID.

## Import

Import with `{property_id}.{data_stream_id}`:

```bash
terraform import googlemarketing_ga4_web_data_stream.example 123456789.987654321
```

The full GA4 Admin name is also accepted, for example `properties/123456789/dataStreams/987654321`.
