# googlemarketing_ga4_custom_metric

Typed GA4 custom metric resource.

Manages parameter name, display name, measurement unit, scope, description, and restricted metric type.

## Import

Import with `{property_id}.{custom_metric_id}`:

```bash
terraform import googlemarketing_ga4_custom_metric.example 123456789.987654321
```

The full GA4 Admin name is also accepted, for example `properties/123456789/customMetrics/987654321`.
